# Job Application Tracker

Personal tool for tracking job applications end-to-end.

See the open PRs for the implementation milestones:

1. **setup** — Vite + TS + Tailwind + Supabase auth + app shell
2. **feature/schema** — Postgres schema with RLS, storage, and TS types
3. **feature/applications** — CRUD, kanban, table, detail, search, archive

Stack: React 18 + Vite + TypeScript, Tailwind, TanStack Query, Supabase (Postgres + Auth + Storage + Edge Functions), deployed on Vercel.
